verified by fetching the live site in June 2026 — but you must still confirm the founder/email

Expand the list yourself via Shopify Partners / Clutch using the recipe in `outreach-kit.md` §2._

## How to work a row
1. Open the **portfolio link**, pick one **client site**.
2. Run A11y Beast on it → fill `templates/evidence-record-template.html` → Print to PDF (their brand on it).
3. Confirm the **founder** + a contact path on LinkedIn (don't guess emails).
4. Send the evidence-record-led message (`outreach-kit.md` §4a/4b). Log in the tracker.

## Caveats — verified vs. what you confirm
- **Verified (June 2026, live fetch):** each site loads and is a genuine small/boutique US web-design agency doing the focus shown, with a working portfolio page. Founder names are listed **only where a real source named them** — none invented.
- **You confirm 3 things per row:** (1) **exact team size** — only Temper & Forge (~10), Big Red Jelly (~16–20), Tag Team (2–10), eSEOspace (~29) were size-confirmed; the rest are "small/boutique/solo" inferred from their language. (2) **the right person to DM** — confirm full name + current role on LinkedIn (several have co-founders; pick the client/marketing-facing one). (3) **email** — none included by design; pull from the site contact form or LinkedIn, never guess a pattern.
- **Lower-fit flags:** eSEOspace (~29, multi-city) and Caveni ("multi-national offices") are the least "founder-reads-DMs" — deprioritize. Aeolidia / Fuel Made / Kinful skew to premium DTC clients but are still founder-led boutiques with small client sites to scan.
- **Sources:** Clutch / DesignRush / GoodFirms + listicles were used only as *candidate leads*; every inclusion was re-verified on the agency's own live site. UpCity is shut down (redirects to Capterra). Dropped for honesty: non-US shops (Studio Elias/Manila, by Crawford/UK, Hatch/Belfast, Visuable/London) and too-large/enterprise ones (Netalico, Freshy, Ambaum, Cosmico). Stopped at 16 solid entries rather than padding to 20.
